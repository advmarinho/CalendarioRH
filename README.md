# Calendário Operacional RH

Sistema web para organização, acompanhamento e controle das rotinas operacionais de RH, DP, Benefícios e Fornecedores ao longo do ano.

Este projeto foi criado para transformar o calendário de obrigações do RH em um **painel vivo de gestão**, permitindo:

- Visualizar todas as atividades do ano em formato de calendário
- Controlar status mensal de cada atividade (feito, pendente, vencido, etc.)
- Centralizar responsabilidades, fornecedores e evidências
- Acompanhar impacto financeiro através de:
  - Valor provisionado
  - Valor executado
  - GAP (diferença entre previsto e realizado)

---

## Objetivo do Projeto

O objetivo é sair do controle disperso em:

- Planilhas
- E-mails
- Lembretes informais
- Memória operacional das pessoas

E passar para um **sistema único, visual e auditável** que:

- Reduz risco operacional
- Aumenta previsibilidade
- Facilita governança
- Cria histórico de execução mês a mês

---

## O que o sistema controla

- Folha de pagamento
- Benefícios (VR, VT, Saúde, Odonto, Previdência, etc.)
- Fornecedores (ADP, SST, Estágio, Terceiros, etc.)
- Atividades mensais, anuais e recorrentes
- Prazos críticos
- Evidências (Pipefy, SSA, etc.)
- Provisionamento x Executado

---

## Principais Funcionalidades

- Visão anual por meses
- Status visual por atividade:
  - Concluído
  - Pendente
  - Vencido
  - Não aplicável
- Filtros por:
  - Categoria
  - Responsável
  - Fornecedor
  - Periodicidade
- Dashboard com:
  - Total de atividades no mês
  - Concluídas, pendentes e vencidas
  - Total provisionado
  - Total executado
  - GAP financeiro
- Salvamento automático no navegador (localStorage)
- Exportação e importação dos dados

---

## Conceito Central

> Este sistema é o **mapa operacional do RH**.

Ele responde perguntas como:

- O que precisa ser feito este mês?
- O que está atrasado?
- O que depende de fornecedor?
- O que gera custo?
- Onde estamos gastando mais do que o previsto?
- Onde existe risco operacional?

---

## Tecnologias

- HTML
- CSS
- JavaScript puro (sem framework)
- Armazenamento local (localStorage)

---

## Uso

1. Abrir o arquivo `index.html` no navegador
2. O sistema carrega automaticamente os dados salvos
3. Marcar atividades, lançar valores e acompanhar os indicadores
4. Exportar os dados para backup ou compartilhamento
